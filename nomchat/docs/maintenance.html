NomChat — Under Maintenance

NomChat is currently undergoing a major rework. The service is temporarily unavailable while we rebuild it from the ground up.

⚠️ Service Unavailable

NomChat is not accepting new connections at this time. All existing sessions have been paused.

What's happening

We are performing a complete architectural overhaul of the NomChat platform. This includes:

  • Full backend rewrite for better performance and reliability
  • New real-time messaging infrastructure
  • Improved NomChat ID integration across all services
  • Enhanced security and privacy features
  • New UI/UX design system

What this means for you

During the maintenance period:

  • NomChat web app is unavailable
  • NomChat ID authentication still works for connected apps (AI Chat Company, hQd Pays)
  • Your account data is safe and will be preserved
  • API endpoints for third-party apps remain operational

Services still available

✅ Still working

NomChat ID login, token verification API, hQd Pays, AI Chat Company, and all documentation remain fully operational.

Expected timeline

We don't have a fixed date yet — we want to get it right rather than rush it. Follow our updates in the docs or check back here for the latest status.
